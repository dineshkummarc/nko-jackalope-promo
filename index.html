Real-time Website Stats

You're viewing a demo site created for Jackalope, a real-time web stats viewing application entry for the Node Knockout competition.

Every movement you make with your mouse is instantly transferred back through our Node.js server and back to the admin view, generating a heatmap in real-time for the admin to view!
